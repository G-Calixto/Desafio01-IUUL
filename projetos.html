PYTHON

Python é uma linguagem de programação de alto nível, versátil e com uma vasta biblioteca padrão que possibilita seu uso em diversas áreas, como desenvolvimento web, automação, análise de dados e aprendizado de máquina.

PRÓS

  • Facilidade de leitura
  • Amplo ecossistema de bibliotecas
  • Comunidade ativa

CONTRAS

  • Mais lento que linguagens compiladas
  • Não ideal para dispositivos móveis
  • Gerenciamento de memória

HTML5

HTML5 é a última versão da linguagem de marcação utilizada para estruturar e exibir conteúdo na web. Ela introduz uma série de novos elementos, atributos e funcionalidades para tornar o desenvolvimento web mais poderoso e eficiente.

PRÓS

  • Responsividade Nativa
  • Estruturação semântica do conteúdo
  • Suporte para vídeos e áudios sem plugins externos.

CONTRAS

  • Requer um navegador moderno para aproveitar ao máximo seus recursos.
  • Não é projetado para manipulação avançada de gráficos
  • Sem suporte nativo para desenvolvimento de aplicativos off-line
w3c/html5

JAVASCRIPT

JavaScript é uma linguagem de programação essencial para a web, responsável por adicionar interatividade, controle de dados e funcionalidades dinâmicas nas páginas.

PRÓS

  • Executado diretamente no navegador, sem necessidade de compilações.
  • Flexível e compatível com inúmeras bibliotecas e frameworks.
  • Oferece manipulação avançada do DOM para interatividade.

CONTRAS

  • Pode gerar inconsistências de desempenho em navegadores diferentes.
  • Em projetos maiores, pode ficar desorganizado sem boas práticas.
  • Pode ser mais vulnerável a ataques como Cross-Site Scripting (XSS)
w3c/javascript

Bootstrap

Bootstrap é um framework front-end desenvolvido para facilitar a criação de layouts responsivos e estilos consistentes. Ele oferece componentes prontos para uso e um sistema de grade robusto.

PRÓS

  • Facilita a criação de layouts responsivos de forma rápida.
  • Inclui uma grande variedade de componentes e estilos prontos.
  • Extensível com temas personalizados e integração com outras tecnologias.

CONTRAS

  • Pode resultar em arquivos CSS mais pesados, se não otimizado.
  • Estilos padrões podem dificultar personalização profunda.
  • Requer atenção para evitar um visual muito “padrão” ou comum.
bootstrap.com
FECHAR

CSS3

CSS3 é a versão mais recente da linguagem de estilos CSS, que define a aparência e o layout de uma página web. Ele permite a criação de interfaces visualmente atraentes e responsivas.

PRÓS

  • Proporciona personalização avançada de layouts com flexibilidade.
  • Inclui transições, animações e transformações 3D.
  • Permite a criação de design responsivo com media queries.

CONTRAS

  • Algumas propriedades são inconsistentes entre navegadores.
  • Não possui controle direto sobre a lógica de interatividade.
  • Manipular CSS em projetos complexos pode se tornar desafiador.
